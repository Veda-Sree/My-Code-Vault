Text Encryption and Decryption

For my second project, I developed a text encryption and decryption application in Python, implementing several classical ciphers including the Caesar cipher, monoalphabetic substitution cipher, Playfair cipher, and rail fence cipher.

The application allows users to encrypt plaintext messages using any of the mentioned ciphers and decrypt ciphertext messages back to their original form. Each cipher has its unique method of encryption and decryption, providing different levels of security and complexity.

Through this project, I gained a deeper understanding of cryptography principles and the mechanics behind classical ciphers. I learned how to implement encryption algorithms using Python, handle input/output operations, and ensure the security of encrypted messages.

This project also served as an opportunity to explore different encryption techniques and compare their strengths and weaknesses in terms of security and efficiency.

Interactive Mini Games in Python

For my third project, I created a collection of interactive mini games using Python. These games were designed to be fun and engaging, providing users with entertainment while also showcasing my programming skills.

The mini games included in this project cover a variety of genres and challenges, offering something for everyone:

  • Rock, Paper, Scissors: A classic game where players compete by choosing rock, paper, or scissors and see who wins based on the rules.
  • Hangman: A word-guessing game where players try to guess the hidden word letter by letter before the hangman is fully drawn.
  • Number Guessing Game: A simple game where players try to guess a randomly generated number within a specified range.
  • Dice Rolling Game: A dice rolling simulation where players roll two dice and guess the total sum of the numbers rolled.

Each game features interactive gameplay elements and intuitive user interfaces, providing an enjoyable experience for players of all ages.

Through this project, I honed my skills in game development, problem-solving, and user interaction design. I learned how to implement game mechanics, handle user input, and create engaging gameplay experiences using Python.
